Tags

Closing Tags

Div - Division, P - Paragraph, H1-H6 - Headings, section - Section, article - Article, nav - Navigation, header - Header, footer - Footer, aside - Sidebar, main - Main content, table - Table, form - Form, ul - Unordered List, ol - Ordered List, li - List Item, blockquote - Block Quotation, pre - Preformatted Text, fieldset - Form group, figure - Figure, figcaption - Figure Caption, address - Contact Information

Self-Closing/Empty Tags

br - Line Break, hr - Horizontal Rule, img - Image, input - Input Field, link - Link, meta - Metadata, area - Image Map Area, base - Base URL, col - Table Column, embed - Embedded Content, param - Object Parameter, source - Media Source, track - Media Track, wbr - Word Break Opportunity

Empty Tags

Empty tags are also known as void elements. They do not have a closing tag and cannot contain any content.
