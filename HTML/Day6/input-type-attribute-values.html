Input Type Attribute Values

  • text: A single-line text input field.
  • password: A single-line text input field that masks the input.
  • email: A single-line text input field for email addresses.
  • number: A single-line text input field for numeric values.
  • checkbox: A checkbox input control.
  • radio: A radio button input control.
  • submit: A button that submits the form.
  • reset: A button that resets the form fields to their default values.
  • file: An input field that allows users to select files from their device.
  • hidden: An input field that is not visible to the user but can hold data to be submitted with the form.
  • date: A single-line text input field for selecting a date.
  • color: A single-line text input field for selecting a color.
  • range: A single-line text input field for selecting a value from a range.
  • tel: A single-line text input field for telephone numbers.
  • url: A single-line text input field for URLs.
  • search: A single-line text input field for search queries.
  • datetime-local: A single-line text input field for selecting a date and time, without a time zone.
  • month: A single-line text input field for selecting a month and year.
  • week: A single-line text input field for selecting a week and year.
  • time: A single-line text input field for selecting a time.
  • datetime: A single-line text input field for selecting a date and time, with a time zone.
