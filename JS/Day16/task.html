Callback Function

A callback function is a function passed as an argument to another function and executed after another function completes. To execute one function after another function finishes
Used in asynchronous operations like:
API calls
Timers
Events

Why Promise?

A Promise is used to handle asynchronous operations and avoid Callback Hell.

Why Async / Await?

async/await is a cleaner way to write Promise code. Looks like normal synchronous code
Easier to understand
Easier error handling
Avoids multiple .then() chains

Why Fetch?

fetch() is used to communicate with APIs. It helps to:
Get data from server
Send data to server
