What is CSS

CSS (Cascading Style Sheets) is used to style and design HTML webpages. It controls colors, fonts, layouts, spacing, animations, and responsiveness.

Why is CSS used?

CSS is used to: Improve webpage appearance Separate content from design Create responsive layouts Add animations and effects

What are the three types of CSS?

Inline CSS Internal CSS External CSS

Which CSS has the highest priority?

Inline CSS Internal CSS External CSS

What is a CSS selector?

A selector targets HTML elements for styling.

What is the Universal Selector?

Selects all elements.

What is a Group Selector?

Styles multiple elements together.

What is an Attribute Selector?

Targets elements based on attributes.

What is the Box Model?

The Box Model consists of: Content Padding Border Margin

What is Flexbox?

A one-dimensional layout system used to align items in rows or columns.

What is Grid?

A two-dimensional layout system for rows and columns.

What is justify-content?

Aligns items horizontally.

What is align-items?

Aligns items vertically.

What is flex-wrap?

Moves items to the next line if needed.

What is Position Property?

Controls element positioning. Types: static relative absolute fixed sticky

What is z-index?

Controls stacking order. Higher value appears on top.

What are Pseudo Classes?

Used for special states. Examples: :hover :focus :active :nth-child()

What are Pseudo Elements?

Used to style specific parts of an element.

What is Transition?

Creates smooth property changes.

What is Transform?

Changes size, position, or rotation.

What is a Media Query?

Makes websites responsive.

What is Responsive Design?

A design that adapts to different screen sizes like mobile, tablet, and desktop.

What is CSS Specificity?

Determines which CSS rule is applied. Priority: Inline ID Class/Attribute/Pseudo-class Element

What is Cascading in CSS?

When multiple styles apply to the same element, CSS decides which one wins based on priority and specificity.

What is Inheritance?

Some CSS properties are inherited from parent elements. Child elements inherit the color.

What is the difference between Visibility and Display?

Element hidden but space remains. Element hidden and space removed.

What is CSS Animation?

Allows movement and visual effects.

What is box-sizing:border-box?

Includes padding and border inside width and height.

What are CSS Variables?

Reusable custom values.

What is the difference between em, rem, %, px?

px Fixed size % Relative to parent em Relative to parent font size rem Relative to root font size

What is the difference between nth-child() and nth-of-type()?

Targets second child. Targets second li element.

What is the difference between visibility:hidden and opacity:0?

visibility:hidden → not visible and not clickable. opacity:0 → invisible but still clickable.

What is a CSS Framework?

A pre-written CSS library. Examples: Bootstrap Tailwind CSS Bulma
